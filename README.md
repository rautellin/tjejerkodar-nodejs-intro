
# Node.js intro


## Run Locally

Clone the project

```bash
  git clone https://github.com/rautellin/tjejerkodar-nodejs-intro/
```

Go to the project directory

```bash
  cd tjejerkodar-nodejs-intro
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
